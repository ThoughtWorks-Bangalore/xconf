Agenda

4:00 - 7:00 pm

Big Data Workshop

We will start with the word count algorithm, and use this to familiarize ourselves with the MapReduce programming model and the deployment infrastructure. Along the way, we'll process a data feed through various tools and try to perform some interesting analysis on that data. At the end, you should have an understanding of how to approach a data engineering problem, and some understanding of the tools in that landscape.

By: Gautam Punhani & Syed Atif Akhtar

Resources: Apache Hadoop
